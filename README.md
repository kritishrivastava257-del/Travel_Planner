Smart Travel Planner is a responsive web application designed to help users plan their trips efficiently by providing essential travel-related information in one place. The application allows users to search for cities and destinations, view real-time weather conditions, explore popular tourist attractions, and estimate travel expenses using a built-in budget calculator. To enhance the user experience, the project integrates multiple REST APIs, including OpenWeather API for weather updates, Unsplash API for destination images, and Geoapify API for location-based information. The application features a modern and intuitive user interface built using HTML, CSS, Tailwind CSS, and JavaScript, ensuring compatibility across different devices and screen sizes. Through dynamic API integration and responsive design principles, Smart Travel Planner offers travelers a convenient platform for discovering destinations and organizing their journeys more effectively
