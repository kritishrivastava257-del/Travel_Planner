Smart Travel Planner is a responsive web application designed to help users plan their trips efficiently by providing essential travel-related information in one place. The application allows users to search for cities and destinations, view real-time weather conditions, explore popular tourist attractions, and estimate travel expenses using a built-in budget calculator.
Tech Stack
HTML5 – Structure and content of the web application.
CSS3 – Styling and layout design.
Tailwind CSS – Responsive and modern user interface development.
JavaScript (ES6+) – Application logic and dynamic functionality.
REST APIs – Fetching and displaying real-time travel data.
OpenWeather API – Real-time weather information.
Geoapify API – Location details and tourist attraction data.
Unsplash API – Destination and travel-related images.
Fetch API – Handling asynchronous API requests.
DOM Manipulation – Dynamic content rendering and user interactions.
Responsive Web Design – Cross-device compatibility and accessibility.
Git & GitHub – Version control and project management.
