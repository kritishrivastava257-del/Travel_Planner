A responsive travel planning web application that helps users explore destinations, check weather conditions, discover tourist attractions, and estimate travel expenses.

Features
🌍 Search any city or destination
☁️ Real-time weather information
🏞️ Tourist attraction recommendations
🖼️ Destination images
💰 Travel budget calculator
📱 Fully responsive design
⚡ Fast and interactive user interface
Tech Stack
HTML5
CSS3
Tailwind CSS
JavaScript
REST APIs
APIs Used
OpenWeather API
Unsplash API
Geoapify API
